# tools

- Este repositória tem como objetivo implementar algumas funcionalidades que são encontradas nos ecossistemas de microsserviços. 
- A maioria das funcionalidades será implementadas utilizando kotlin e spring. 

## O que contém

- Documentação com c4Model utilizando o PlantUml
- Exemplos de métricas utilizando Spring Sleuth e Zipkin
- Exemplo de configuração do sistema de coleta e exibição de logs utilizando Elastic Stack
